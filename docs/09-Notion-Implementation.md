# Critério de Aceite

O Operation Backend será considerado pronto quando:

- Todo o fluxo funcionar apenas pelo Dashboard.
- O usuário souber a próxima missão em menos de 1 minuto.
- Todo progresso estiver conectado entre as databases.
- O sistema exigir menos de 5 minutos diários de organização.
