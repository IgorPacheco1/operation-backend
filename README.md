# 📖 Sobre o Projeto

O **Operation Backend** é um sistema de produtividade desenvolvido utilizando o **Notion** como plataforma principal.

Seu objetivo é transformar a preparação para vagas de Desenvolvimento Backend em um processo estruturado, organizado e orientado por resultados.

O projeto foi criado para eliminar a dúvida constante de **"o que estudar agora?"**, transformando um objetivo profissional em uma jornada clara de evolução.

Mais do que um organizador de tarefas, o Operation Backend funciona como um verdadeiro **Sistema Operacional de Evolução Profissional**, centralizando planejamento, estudos, projetos, acompanhamento de progresso e preparação para processos seletivos.

---

# 🎯 Objetivo Principal

Preparar o usuário para conquistar uma vaga de estágio em Desenvolvimento Backend, com foco inicial no **Banco Inter**.

Para isso, o sistema organiza toda a jornada em:

- Roadmap de estudos
- Competências
- Tecnologias
- Missões
- Projetos práticos
- Inglês técnico
- Certificações
- Dashboard de produtividade
- Diário de Bordo
